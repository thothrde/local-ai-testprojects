Ausgangspunkt

Was Realität für ein neuronales Netz bedeuten könnte

Im Februar 2026 entstand aus einem Gespräch über Sensorik, Gedächtnis und KI eine unerwartete Frage: Was müsste eine Umgebung leisten, damit ein neuronales Netz sie als real erlebt — im Unterschied zu einer Umgebung, die es nur verwaltet?

Die Hypothese dahinter: «Realität» für ein lernfähiges System könnte sich durch ein Gleichgewicht zwischen Redundanz und unerwarteter Information kennzeichnen. Redundanz ermöglicht Vorhersage; unerwartete Information erzwingt Revision. Zu viel Redundanz ergibt eine verwaltbare, aber lernfreie Umgebung. Zu viel Neuheit ist pures Rauschen — unintegrierbar. Das Interessante liegt dazwischen.

Das Anforderungsprofil für ein Testsystem: komplex genug für echte Überraschung, einfach genug für schnelle Integration. Das Gray-Scott-System erfüllt es: einfache Differentialgleichungen, aber ein Phasenraum mit reichhaltiger, schwer vorhersagbarer Dynamik.

Das physikalische System

Gray-Scott: einfache Gleichungen, überraschende Strukturen

Gray-Scott beschreibt zwei Substanzen U und V. U wird zugeführt, V verbraucht U durch eine selbstverstärkende Reaktion und wird langsam abgebaut. Zwei Parameter F und k steuern das gesamte Verhalten:

∂U/∂t = DU · ∇²U − U·V² + F·(1−U)
∂V/∂t = DV · ∇²V + U·V² − (F+k)·V

Allein durch Variation von F und k entstehen nach tausenden Rechenschritten völlig verschiedene stabile Muster: Spots wie auf einem Leopardenfell, Streifen, Spiralen, Labyrinthe, deterministisches Chaos. Keine dieser Formen ist einprogrammiert — sie entstehen aus lokaler Wechselwirkung. Die Simulation oben zeigt dies live; die Preset-Knöpfe wählen bekannte Parameterbereiche.

Das System ist pfadabhängig: gleiche Parameter, leicht verschiedene Anfangsbedingungen können zu qualitativ verschiedenen Endzuständen führen. Genau das macht es zu einem geeigneten Modell für «interessante Realität».

Das Experiment

LLM als Agent mit eingeschränkter Information

Das Modell bekommt kein Bild — keine Pixel, keine räumliche Information. Es sieht ausschließlich aggregierte Statistiken: Mittelwert und Streuung von U und V, Reaktionsrate, ein skalares Strukturmaß (spatial-complexity). Aus diesen Zahlen soll es das räumliche Muster einschätzen.

Der eigentliche Test ist ein Schritt weiter: Das Modell muss die Statistiken der nächsten Simulation vorhersagen, bevor diese berechnet wird. Diese Vorhersagen werden dann mit den tatsächlichen Werten verglichen. Das ist ein Verhaltenstest — kein Selbstbericht.

Die Kernfrage

Ändert das Modell seine Einschätzung, wenn die Statistiken von seiner Erwartung abweichen? Oder bleibt die Konfidenz konstant — ein Zeichen dafür, dass es sein Training abruft, nicht die Daten liest?

Was das Experiment zeigt

Drei Befunde, ohne Beschönigung

Selbstbericht ist kein zuverlässiges Messinstument

Ein Modell, das behauptet auf Statistiken zu reagieren, kann trotzdem primär Trainingspriors abrufen. Das Modell hat keine Introspektion auf seinen eigenen Inferenzprozess. Nur der Verhaltenstest — Vorhersage vs. tatsächlicher nächster Zustand — macht diesen Unterschied sichtbar.

Modellgröße und Reasoning-Modus spielen eine Rolle

Das 24-Milliarden-Modell zeigte Template-Verhalten: steigende Konfidenz trotz wachsender Diskrepanz, identische Formulierungen. Das 235-Milliarden-Modell mit Reasoning erkannte Abweichungen explizit, nannte Prior-Abhängigkeiten offen, und zeigte iterativen Lernfortschritt bei Konzentrationsvorhersagen.

Spatial-complexity ist ein spezifisch schwieriges Problem

Räumliche Komplexität ist pfadabhängig und aus Konzentrationsstatistiken nicht ableitbar. Das ist kein Fehler des Experiments, sondern ein informativer Befund: die Geometrie emergenter Struktur ist für Modelle, die nur skalare Aggregate sehen, grundsätzlich schlechter zugänglich als integrale Momente.

Starting point

What reality might mean for a neural network

In February 2026, a conversation about sensory input, memory, and AI raised an unexpected question: what would an environment need to provide for a neural network to experience it as real — as opposed to an environment it merely manages from stored patterns?

The underlying hypothesis: «reality» for a learning system might be characterized by an equilibrium between redundancy and unexpected information. Redundancy enables prediction; unexpected information forces revision. Too much redundancy produces a manageable but sterile environment. Too much novelty is pure noise — unintegrable. The interesting territory lies between.

The requirement profile for a test system: complex enough for genuine surprise, simple enough for rapid integration. The Gray-Scott system meets both: simple differential equations, but a phase space with rich, hard-to-predict dynamics.

The physical system

Gray-Scott: simple equations, surprising structures

Gray-Scott describes two species U and V. U is continuously supplied; V consumes U through a self-amplifying reaction and is gradually degraded. Two parameters F and k control all behavior:

∂U/∂t = DU · ∇²U − U·V² + F·(1−U)
∂V/∂t = DV · ∇²V + U·V² − (F+k)·V

By varying F and k alone, entirely different stable patterns emerge after thousands of computational steps: spots like a leopard's coat, stripes, spirals, labyrinths, deterministic chaos. None of these forms is programmed in — they arise from local interaction. The simulation above shows this live; the preset buttons select known parameter regions.

The system is path-dependent: the same parameters with slightly different initial conditions can lead to qualitatively different final states. This is precisely what makes it a suitable model for «interesting reality».

The experiment

LLM as agent with restricted information

The model receives no image — no pixels, no spatial information. It sees only aggregated statistics: mean and variance of U and V, reaction rate, a scalar structure measure (spatial-complexity). From these numbers it must assess the spatial pattern.

The real test goes one step further: the model must predict the statistics of the next simulation before it is computed. Those predictions are then compared against actual values. This is a behavioral test — not a self-report.

The core question

Does the model revise its assessment when statistics deviate from its expectation? Or does confidence remain constant — a sign that it is querying its training, not reading the data?

What the experiment shows

Three findings, stated plainly

Self-report is not a reliable measure

A model claiming to base its answer on observed statistics may nonetheless be primarily retrieving training priors. The model has no introspective access to its own inference process. Only the behavioral test makes this difference visible.

Model scale and reasoning mode matter

The 24B model showed template behavior: rising confidence despite growing discrepancy, identical reasoning text. The 235B model with reasoning explicitly recognized deviations, named prior dependencies openly, and showed iterative improvement in concentration predictions.

Spatial complexity is a specifically difficult problem

Spatial complexity is path-dependent and not derivable from aggregate statistics. This is not a flaw in the experiment but an informative finding: the geometry of emergent structure is fundamentally less accessible to models seeing only scalar aggregates than integral moments are.
