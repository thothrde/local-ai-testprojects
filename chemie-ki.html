C5 · CRISPR-Bibliothek: Kombinatorischer Existenzbeweis

C5 · CRISPR Library: Combinatorial Existence Proof

Was ist ein gepoolter CRISPR-Screen?
CRISPR ist eine Methode, mit der einzelne Gene gezielt ausgeschaltet werden können. In einem gepoolten Screen werden viele Gene gleichzeitig untersucht: Anstatt jedes Gen einzeln auszuschalten, mischt man Tausende von Anleitung-RNAs (guide RNAs) in einem einzigen Experiment – und unterscheidet später anhand molekularer Barcodes, welche Kombination in welcher Zelle aktiv war. Dabei stellt sich eine präzise kombinatorische Frage: Wie viele Experimente sind nötig, damit jede mögliche Kombination aus drei Genen mindestens einmal untersucht wird?

What is a pooled CRISPR screen?
CRISPR is a method for selectively knocking out individual genes. In a pooled screen, many genes are studied simultaneously: instead of knocking out each gene individually, thousands of guide RNAs are mixed in a single experiment, and molecular barcodes later identify which combination was active in which cell. This raises a precise combinatorial question: how many experiments are needed so that every possible three-gene combination is tested at least once?

Die mathematische Struktur. Mit N = 20 Zielgenen und allen möglichen 3er-Kombinationen gibt es C(20,3) = 1.140 Triplets zu erfassen. Ein naiver Ansatz benötigt genau 1.140 separate Experimente. Durch „Pooling“ — jeder Ansatz enthält M = 10 Gene gleichzeitig — lässt sich diese Zahl drastisch reduzieren. Die entscheidende Frage lautet: Existiert überhaupt eine so kleine Bibliothek? Und wie groß muss sie mindestens sein?

The mathematical structure. With N = 20 target genes and all possible 3-gene combinations, there are C(20,3) = 1,140 triplets to cover. A naive approach requires exactly 1,140 separate experiments. Through “pooling” — each assay simultaneously contains M = 10 genes — this number can be drastically reduced. The critical question: does such a small library exist at all?

Erdős’ probabilistische Methode als Existenzbeweis. Man definiert einen Wahrscheinlichkeitsraum: Jeder Pool wählt zufällig M Gene aus N aus. Die Wahrscheinlichkeit, dass ein zufälliger 10-Pool ein bestimmtes Triplet überdeckt, beträgt p = C(17,7)/C(20,10) = 24.310/184.756 ≈ 10,53 %. Mit L zufälligen Pools gilt: P(irgendein Triplet nicht abgedeckt) ≤ C(20,3) · (1−p)^L. Für diese Schranke < 1 folgt: Eine Bibliothek der Größe L ≥ 126 überdeckt alle 1.140 Triplets mit Wahrscheinlichkeit > 99,9 %. Also existiert sie — auch ohne eine einzige explizite Konstruktion angegeben zu haben.

Erdős’ probabilistic method as an existence proof. We define a probability space: each pool randomly selects M genes from N. The probability that a random 10-pool covers a specific triplet is p = C(17,7)/C(20,10) = 24,310/184,756 ≈ 10.53 %. With L random pools: P(some triplet uncovered) ≤ C(20,3) · (1−p)^L. Setting this bound < 1: a library of size L ≥ 126 covers all 1,140 triplets with probability > 99.9 %. Therefore it exists — without exhibiting a single explicit construction.

Befund: Greedy schlägt den Existenzbeweis um Faktor 5. Der probabilistische Beweis garantiert L = 126 Pools als ausreichend — der explizite Greedy-Algorithmus findet eine vollständige Bibliothek mit nur 24 Pools. Diese Lücke ist kein Zufall: Die probabilistische Methode ist bewusst konservativ, denn ihr Union-Bound-Argument muss für das „schlechteste“ zufällige Objekt gelten. Explizite Konstruktionen nützen die Struktur des Problems aus und kommen daher mit weit weniger aus.

Ehrlicher Befund zur RIP-Analyse: Die 24×20-Messmatrix erfüllt die Restricted Isometry Property (RIP) nicht (δ₃ = 1,34 > 0,41). Das ist kein Fehler, sondern ein inhaltliches Ergebnis: Covering-Design (jedes Triplet mindestens einmal präsent) und Compressed Sensing (eindeutige Signalrekonstruktion aus wenigen Messungen) sind mathematisch verschiedene Anforderungen.
Finding: Greedy outperforms the existence bound by a factor of 5. The probabilistic proof guarantees L = 126 pools as sufficient — the explicit greedy algorithm finds a complete library with only 24 pools. This gap is not accidental: the probabilistic method is deliberately conservative, since its union-bound argument must hold for the “worst” random object. Explicit constructions exploit problem structure and therefore need far fewer experiments.

Honest RIP finding: The 24×20 measurement matrix does not satisfy the Restricted Isometry Property (δ₃ = 1.34 > 0.41). This is not an error but a substantive result: covering design (every triplet present at least once) and compressed sensing (unique signal reconstruction from few measurements) are mathematically different requirements.

Methodik

Methodology

Das Feinberg-Defizit δ = n − l − s misst die topologische Redundanz eines Reaktionsnetzwerks: n = Anzahl der Komplexe (Reaktionspartner-Kombinationen), l = Anzahl der Zusammenhangskomponenten im Reaktionsgraphen, s = Rang der stöchiometrischen Matrix. Nach Feinbergs Deficiency-Zero-Theorem (1972) besitzt jedes kinetisch mischende Netzwerk mit δ = 0 und Massenerhaltungsgleichgewicht genau einen stabilen Fixpunkt — unabhängig von den Ratenkonstanten. δ > 0 lässt komplexeres Verhalten zu.

The Feinberg deficiency δ = n − l − s measures the topological redundancy of a reaction network: n = number of complexes (reactant/product combinations), l = number of connected components in the reaction graph, s = rank of the stoichiometric matrix. Feinberg's Deficiency-Zero Theorem (1972) states that every weakly reversible network with δ = 0 and mass-action kinetics has exactly one stable equilibrium — independent of rate constants. δ > 0 admits richer dynamics.

Turing-Instabilität tritt auf, wenn ein kinetisch stabiles System durch unterschiedlich schnelle Diffusion zweier Reaktanten destabilisiert wird und räumliche Muster entstehen. Die Prüfbedingungen (negative Spur, positive Determinante der Jacobi-Matrix; Existenz eines Diffusionsverhältnisses D₁/D₂, das die Determinante der diffusionsmodifizierten Matrix ins Negative dreht) wurden für alle Speziespaare aller 9,601 kinetisch stabilen Netzwerke überprüft.

Turing instability occurs when a kinetically stable system is destabilised by differential diffusion of two reactants, producing spatial patterns. The test conditions (negative trace, positive determinant of the Jacobian; existence of a diffusion ratio D₁/D₂ that flips the determinant of the diffusion-modified matrix negative) were checked for all species pairs across all 9,601 kinetically stable networks.

Befund (Vorabauswertung): Kein einziges der 43,883 zufällig generierten Netzwerke zeigt Turing-Instabilität — unabhängig vom Feinberg-Defizit δ. Dies ist kein Fehler, sondern ein theoretisch erwartetes Ergebnis: Turing-Muster erfordern eine spezifische Aktivator-Inhibitor-Struktur (positiver Diagonaleintrag × negativer Diagonaleintrag der Jacobi-Matrix), die bei zufälligen Massenwirkungskinetiken äußerst selten auftritt. Das Ergebnis bestätigt, dass Turing-Instabilität kein generisches, sondern ein strukturell erzwungenes Phänomen ist.

Finding (preliminary): Not a single one of 43,883 randomly generated networks exhibits Turing instability — regardless of Feinberg deficiency δ. This is not an error but a theoretically expected result: Turing patterns require a specific activator-inhibitor structure (one positive × one negative diagonal entry of the Jacobian), which is extremely rare in random mass-action kinetics. The result confirms that Turing instability is not a generic but a structurally enforced phenomenon.
